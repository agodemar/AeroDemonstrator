API Reference
=============

The ``aerodemo`` package provides Python classes and functions for
aerodynamic demonstrations.

Module Overview
---------------

The three modules below cover the core functionality of the ``aerodemo`` package:

- :mod:`aerodemo.naca_airfoil` — NACA 4- and 5-digit airfoil geometry and thin-airfoil theory
- :mod:`aerodemo.vlm` — Vortex Lattice Method solver for finite wings
- :mod:`aerodemo.openvsp_utils` — OpenVSP geometry creation and VSPAero analysis helpers
